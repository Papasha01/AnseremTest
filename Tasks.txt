
Для решения задания вам потребуется установить node.js (обычно устанавливают последней версией) и VisualStudio для задания на C#
js файл запускается через терминал в VSCode или через node.js comand promt командой node root.js
в момент запуска node.js должна быть открыта в дирректории проекта, иначе допишите путь к названию файла
c# проект запускается из solution под debug, этого будет достаточно
сначала запускаете c# проект, затем фронт проект
за 1 итерацию js проекта проходит полный цикл задания

test.json - файл для задания

решение можете также отправить архивом, полученные данные разместите в архиве в текстовом файле вместе с решением.

JS
- Все, кто имеет поле isActive = true, отправить на сервер для сбора статистики
- ответ записать в тестовый файл, название файла любое

С#
- в ответе с сервера присылать следующие данные:
    Имя
    Количество Друзей
    Номер телефона (
        если номер телефона не проходит по маске или содержит меньше 11 цифр,
        то в номер телефона записать сообщение об ошибке
        )
    E-mail (Сделать по анологии с номером телефона)
    У каждого человека есть список друзей, 
    образует ли он с кем то из них дружескую пару 
    (является ли он в друзьях одного из своих друзей)


формат записи ответа

- пример с положительными данными
Имя: Colon Salazar
Номер телефона: +1 (952) 533-5657
Почта: logankeller@artiq.com
Количество друзей: 3
Дружеские пары: Colon Salazar - Carol Martin, Colon Salazar - Rogan Keller

- пример с отрицательными данными (варианты сообщений можете придумать свои)
Имя: French Mcneil
Номер телефона: "Номер телефона содержит ошибки"
Почта: "Почта содержит ошибки"
Количество друзей: 3
Дружеские пары: "Дружеские пары отсутствуют"
